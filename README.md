# 🌐 Personal Portfolio Website

## 📌 Overview

This is a responsive and modern personal portfolio website developed using **HTML5** and **CSS3**. It showcases my educational background, technical skills, career objective, hobbies, and contact information in a clean and professional layout. The website is designed to provide visitors with a quick overview of my profile as an aspiring Computer Science Engineer.

## ✨ Features

* Modern and responsive user interface
* Professional landing (Hero) section
* About Me section
* Education timeline
* Technical skills with progress bars
* Soft skills section
* Hobbies section
* Contact information
* Smooth scrolling navigation
* Attractive animations and hover effects
* Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Responsive Web Design

## 📂 Project Structure

```
Portfolio/
│── index.html
│── style.css
└── README.md
```

## 🎯 Purpose

The purpose of this portfolio is to present my academic achievements, technical skills, and personal interests in a professional manner. It serves as an online resume that can be shared with recruiters, educators, and potential employers.
